Windows常用Tips

2019年5月6日
| Misc
| 阅读

1. 常用快捷键

  • win10自带输入法切换简体繁体快捷键: Ctrl+Shift+F
  • Win10自带的截图工具: Win+Shift+S
  • 快速关闭窗口和计算机：Alt + F4
  • 快速回到桌面 Win + D
  • 快速打开运行 Win + R
  • 切换窗口：Alt + Tab
  • 任务视图：Win + Tab(松开键盘界面不会消失)
  • 任务管理器： Ctrl + Alt + Del
  • 查看win10激活信息：slmgr.vbs -dlv
  • 查看win10是否永久激活：slmgr.vbs -xpr
  • 快速打开服务：services.msc
  • 快速打开注册表编辑器：regedit
  • 快速打开虚拟键盘：osk
  • 设置定时关机：Shutdown -s -t 3600 （3600s后关机）

2. bat文件的编写

在cmd窗口中：for %I in (command1) do command2 
在批处理文件中：for %%I in (command1) do command2

for /l %x in (1, 1, 118) do ffmpeg.exe -i %x.mp4 -r 10 -c:v libx264 -crf 28 -preset veryfast -c:a copy -movflags +faststart %x0.mp4 -y

比如合并B站视频:

@echo off

for /l %%x in (1, 1, 118) do ffmpeg.exe -i ".\%%x\64\video.m4s" -i ".\%%x\64\audio.m4s" -vcodec copy -acodec copy -f mp4 "%%x.mp4"

pause

压缩视频:

@echo off & setlocal EnableDelayedExpansion
for /f "delims=" %%i in ('"dir /a/s/b/on *.mp4"') do (
set file=%%~fi
set file=!file:/=/!
ffmpeg.exe -i !file! -r 10 -c:v libx264 -crf 28 -preset veryfast -c:a copy -movflags +faststart !file!_cp.mp4 -y
) 

pause

抽取视频中的音频:

for %i in (*.mp4) do ffmpeg -i "%i" "%i.mp3"

3. 关闭windows自动修复模式

bcdedit /set {current} recoveryenabled No

4. 右键管理员权限打开cmd窗口

将下列内容保存到.reg文件中。

Windows Registry Editor Version 5.00
[HKEY_CLASSES_ROOT\Directory\shell\runas]
@="Open cmd here as Admin"
"HasLUAShield"=""
[HKEY_CLASSES_ROOT\Directory\shell\runas\command]
@="cmd.exe /s /k pushd \"%V\""
[-HKEY_CLASSES_ROOT\Directory\Background\shell\runas]
[HKEY_CLASSES_ROOT\Directory\Background\shell\runas]
@="Open cmd here as Admin"
"HasLUAShield"=""
[HKEY_CLASSES_ROOT\Directory\Background\shell\runas\command]
@="cmd.exe /s /k pushd \"%V\""
[-HKEY_CLASSES_ROOT\Drive\shell\runas]
[HKEY_CLASSES_ROOT\Drive\shell\runas]
@="Open cmd here as Admin"
"HasLUAShield"=""
[HKEY_CLASSES_ROOT\Drive\shell\runas\command]

5. 开机启动Remote Debugger

建立一个bat文件，里面写

cd /d %~dp0
%1 start "" mshta vbscript:createobject("shell.application").shellexecute("""%~0""","::",,"runas",1)(window.close)&exit
pushd D:\Remote Debugger\x64
start /min msvsmon.exe /noauth /nosecuritywarn /anyuser /timeout 9000000000

C:\Users\Administrator\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
