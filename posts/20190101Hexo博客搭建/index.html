Hexo博客搭建

2019年1月1日
| 阅读

本文用来记录我使用Hexo进行静态博客搭建的过程，博客是托管在Github Page上的。

1.下载安装node.js和git两个工具

因为需要使用npm和git进行博客的生成，搭建和上传部署。

npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server

运行完成上面的命令之后就可以看到cmd窗口提示已经将blog发布到了localhost:4000上，然后在浏览器中打开这个链接就可以看到初始的一个博客面貌了。

2. 在github上创建代码仓库

创建一个新的代码仓库，代码仓库需要命名为：username.github.io, 其中的username就是你的github帐户名。 这里一定要注意这个命名，很容易出错。

3. 修改博客配置

博客的配置文件在hexo init后的根目录下面，名字是_config.yml。 在deploy中进行如下方式的修改。type选用git, repo这里就是你的blog repo的ssh地址。branch是blog这个repo的分支，一般只有一个主分支，所以直接使用master就可以了。

deploy:
  type: git
  repo: git@github.com:bzhou830/bzhou830.github.io.git
  branch: master

修改完成之后进行保存。这里需要注意的是这里的每一项冒号后面都要有一个空格。

4. 上传

在cmd窗口下使用如下的命令进行上传：

hexo g -d

上面的命令中， g是generater, d是deploy。 至此，在浏览器中使用https://bzhou830.github.io/ 就可以访问到你的博客了，当然bzhou830这个需要换成你自己的github用户名。
