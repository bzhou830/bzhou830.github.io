关于本博客

　　[2020.04.16更新] 购买的阿里云服务器到期了，考虑在云服务器费用问题，而且访问量好像也很低，所以不打算续费了。目前直接搭载到gitee page上，域名binean.top还在有效期内。

　　[2018.08.06更新] 写博客的初衷是记录学习笔记方便以后查找，以及记录自己在工作和生活中的心路历程。最初是在CSDN上我原来的博客，考虑自己搭建博客内容更容易保存和迁移，所以使用git page搭建了博客，在云服务器打折的时候购买了云主机和域名，然后就搭建了这个独立博客。起初使用的是hexo + Github page部署，后来发现hugo编译静态博客速度快，而且依赖的插件少，对于不懂前端的人来说更加友好，所以使用了hugo。主题是在飞雪无情改写maupassant主题上进行改造的。博客中讨论的任何内容都欢迎留言指导，交流。

关于我

操千曲而后晓声, 观千剑而后识器。 ——刘勰《文心雕龙》

　　自幼爱好电子小制作，大学就读电气工程及其自动化专业。然而业余爱好电子设计，所以本科期间常泡在电子技术协会，和一群爱好电子制作的小伙伴一起学习单片机嵌入式相关知识，和小伙伴们参加过各类大学生电子设计竞赛，也拿过各种电子设计竞赛的奖项，期间主要负责单片机编程。

　　因2012年全国大学生电子设计竞赛期间接触并使用使用单片机摄像头进行图像处理，感觉图像处理领域高端大气上档次，遂考研到电子科技大学在资深医学图像处理导师门下读研，期间在导师公司参与华西医院PACS系统开发，并在业余使用OpenCV + DCMTK + VTK进行肺结节图像检测识别研究，后作为硕士毕业课题。项目代码

　　毕业至今，定居武汉，就职于国内IC企业，主要从事Windows系统下GPU驱动相关工作。工作之余喜欢和几个好友一起在城市周边骑行。早在上学期间开了微信公众号，不定期的更新一些文章在上面，欢迎关注。
